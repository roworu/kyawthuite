## kyawthuite
kyawthuite - minimal and stable desktop OS based on Fedora, Bazzite and Universal Blue technologies

kyawthuite (pronounced as "kyatuit") - named after the [Kyawthuite](https://wikipedia.org/wiki/Kyawthuite) mineral, a rare natural bismuth antimonate (BiSbOâ‚„) known for its **stability and distinct crystalline structure**.

### system tools

to check for system updates (system image + flatpak packages + brew packages):
```bash
sys-update
```

to upgrade system:
```bash
sys-upgrade
```

to upgrade and reboot system:
```bash
sys-upgrade-reboot
```
